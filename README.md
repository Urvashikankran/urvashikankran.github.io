## Featured Publication

### A Deterministic Kinetic Framework for Modelling Pathogen Persistence and Recurrence in Zoonotic Reservoirs

This independent research introduces a deterministic mathematical framework inspired by chemical kinetics to investigate pathogen persistence within zoonotic reservoirs. The project integrates mathematical modeling and Python-based simulations to analyze the effects of environmental parameters on disease recurrence.

📄 **DOI:** https://doi.org/10.5281/zenodo.20492904
